Como ver os Sólidos?

  • Com o botão esquerdo do rato arrasta os sólidos para os mover.

  • Para fazer zoom usa o mesmo procedimento anterior mas com a tecla Shift pressionada.

  • Para fazer efeito stereo pressiona a tecla Ctrl ao mesmo tempo que usas o rato.

  • Para desfazer partes dos sólidos pressiona o botão direito do rato e move-o verticalmente.
